# Scoreboard

A simple scoreboard web app built with JavaScript to practice DOM updates, state and user interaction.

## Live Demo
https://scorekeeper-game-app.netlify.app/

## Features
- Increase home and guest scores
- Reset scores
- Fast UI updates driven by JavaScript

## Built With
- JavaScript (ES6)
- HTML
- CSS

## What I Learned
- Managing basic state (scores) in JavaScript
- Updating the DOM in response to button clicks
- Keeping UI and logic in sync

## Next Improvements
- Add period/quarter tracking and a game timer
- Add keyboard shortcuts
- Improve accessibility (button labels, focus states)
